3D AR Model viewer


Loading 3D Models

A 3D model is loaded and displayed above a Hiro marker. To upload your own model, you need the export it to "mtl" - texture and color file and "obj" - file that holds the geometry of your model. There is an ability to call mockup model for preview purposes.


To be able to visualize the model, download the Hiro marker and print it or display it on a portable display.

Web based model augmented reality viewer. By Michał Hęćka based on AR-Examples by stemkoski and FileSaver by Eli Grey, eligrey.com.
